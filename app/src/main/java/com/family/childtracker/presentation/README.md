# Presentation Layer

This layer contains:
- **UI Components**: Jetpack Compose screens and composables
- **ViewModels**: UI state management and business logic coordination
- **Navigation**: Navigation graph and routes
- **Theme**: Material Design 3 theming

## Structure
```
presentation/
├── screen/
│   ├── dashboard/      # Dashboard screen
│   ├── profile/        # Child profile screens
│   ├── growth/         # Growth tracking screens
│   ├── milestone/      # Milestone screens
│   ├── behavior/       # Behavior tracking screens
│   ├── tips/           # Parenting tips screens
│   ├── chat/           # AI chat screens
│   ├── summary/        # Weekly summary screens
│   └── settings/       # Settings screens
├── component/          # Reusable UI components
├── navigation/         # Navigation setup
└── theme/              # Material Design theme
```
